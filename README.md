# Mills Electrical Services 

An official website created for Samuel Mills services based at Muncie, Indiana. 

# Built with

- ReactJS
- Smooth Scroll
- TailwindCSS

Deployed using Netlify. Domain bought with Name.

# Tested with

- Enzyme
- Jest
- LightHouse

# View locally

- Fork the repository
- Run the file on localhost.

`npm start`
